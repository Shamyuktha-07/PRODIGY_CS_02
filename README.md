# PRODIGY_CS_02
"Simple Image Encryption Tool using Pixel Manipulation in Python."

## Description
A simple Python tool that encrypts and decrypts images using pixel manipulation. It utilizes the XOR mathematical operation on RGB values to scramble image data into an unreadable format.

## Technologies Used
* Python 3.x
* Pillow (PIL) Library

## How to Run
1. Clone the repository:
   `git clone https://github.com/Shamyuktha-07/PRODIGY_CS_02.git`
2. Install dependencies:
   `pip install Pillow`
3. Run the script:
   `python encrypt_tool.py`

## Features
* **Encryption:** Scrambles pixels using a secret numeric key.
* **Decryption:** Restores the original image using the same key.
* **Lossless Security:** Uses .png format to ensure zero data loss during the process.
